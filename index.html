Die LabRes Open Source Infrastruktur verbindet Bürger, Test-Center, Labore und Entwickler und machto so digitale Lösungen im Kampf gegen Covid-19 möglich

  • Maximale Transparenz

    Die LabRes Infrastruktur basiert ausschließlich auf Open Source Software Komponten. Der Code kann wird auf Github unter der MIT Licence veröffentlicht. Die LabRes Infrastruktur wird den Laboren kostenfrei zur Verfügung gestellt und zentral in einem AWS Datencenter in Frankfurt in einem High Availablity Setup betrieben.

  • Hohe Sicherheit durch Anonymität

    LabRes verarbeitet keine personenbezogenen Daten. Registrierte Entwicklerungspartner müssen nachweisen, dass sie Prozesse zum verantwortungsvollen Umgang mit Gesundheitsdaten etabliert haben und garantieren, dass Sie diese Gesundheitsdaten nicht mit personenbezogenen Daten verknüpfen.

  • Einfache Anbindung

    Labore können sich sehr einfach an die LabRes Infrastruktur anbinden. Wir unterstützen out-of-the box die gängigen Laborformate wie LDT und HL7/OBX und können laborspezifische Formate schnell umsetzen. Über das gängige HTTPS Protokoll wird mit Hilfe von Zertifikaten, IP Whitelisting, Client Key und Secrets eine sichere Verbindung hergestellt. Unsere Experten helfen gerne bei der Integration.

  • Flexible Prozesse

    Laborbefund und Patient werden über eine Auftrags-ID zusammengebracht. LabRes unterstützt hier grundsätzlich zwei Varianten:

    • Vorgedruckte bzw. generierte Labor spezifische Nummern, die vom Labor normalerweise in Form von Bar oder QR Codes vorgegeben werden und z.B. via App gescannt werden können.
    • Zentral von der LabRes Infrastuktur vergebene 10 Stellige Auftrags-ID, die als externe Auftrags-ID händisch in ein Order Entry System eingegeben oder auf den Muster 10-Überweisungsschein übertragen wird.

  • Test-Center Verzeichnis

    Über das Test-Center Verzeichnis von niederlgelassenen Ärzten, Krankenhäusern und Drive-Ins können Bürger schnell und unkompliziert eine Test-Center in ihrer Nähe finden.
